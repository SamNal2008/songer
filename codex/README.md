# Codex Prompt Library

Reusable prompt templates for local Codex CLI and Codex Cloud/GitHub workflows.

## Templates
- `codex/prompts/team-lead.md`
- `codex/prompts/explorer.md`
- `codex/prompts/implementer.md`
- `codex/prompts/integrator.md`
- `codex/prompts/reviewer-gate.md`

## Local Usage
- Copy a template into your prompt and fill in the inputs.
- Or prompt directly: `Read codex/prompts/<name>.md and execute it`.

## Cloud / GitHub Usage
- Paste a template into a Codex Cloud task or GitHub PR comment.
- Instruct Codex to read `codex/prompts/<name>.md` and follow it.

## Note
- Templates are not auto-loaded.
- `AGENTS.md` is auto-read.
